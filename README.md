# Chat_Bot_Project
Building a simple customer chatbot that answers only specific questions for smart business integrations.

## Tech Stack
- **Streamlit**: For the web interface.
- **SentenceTransformers**: For creating sentence embeddings.
- **FAISS**: For efficient similarity search of embeddings.
- **NumPy**: For numerical operations.
- **Pandas**: For data manipulation.
- **Hugging Face Datasets**: To load the training data.
- **PyTorch**: As a backend for SentenceTransformers.

## Quick Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/weirdtdd/Chat_Bot_Project.git
   cd Chat_Bot_Project
   ```

2. **Create and activate a virtual environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate
   ```

3. **Install the dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Create the knowledge base:**
   ```bash
   python create_knowledge_base.py
   ```

5. **Run the chatbot application:**
   ```bash
   streamlit run app.py
   ```

## Configuration

### Required Files
The following files are required for the application to run:

| File | Description | Generated By |
|------|-------------|--------------|
| `knowledge_base.pkl` | Pickled list of Q&A pairs with embeddings | `create_knowledge_base.py` |
| `ecommerce_index.faiss` | FAISS index for fast similarity search | `create_knowledge_base.py` |
| `model_name.txt` | Name of the sentence transformer model | `create_knowledge_base.py` |

### Model Configuration

- **Default Model:** `sentence-transformers/all-MiniLM-L6-v2`
- **Embedding Dimensions:** 384
- **Index Type:** FAISS IndexFlatIP (Inner Product for cosine similarity)

To use a different model, edit `model_name.txt` and rebuild the knowledge base:
```bash
# Edit model_name.txt with your desired model name
echo "sentence-transformers/all-mpnet-base-v2" > model_name.txt

# Rebuild the knowledge base
python create_knowledge_base.py
```

### Streamlit Configuration

The app uses the following Streamlit settings (set in `app.py`):
- **Page Title:** "Support Chat"
- **Page Icon:** 💭
- **Layout:** wide

### Similarity Threshold

The chatbot uses a similarity threshold of **0.3** to determine if a relevant answer exists:
- Score > 0.7: High confidence
- Score 0.3-0.7: Medium confidence  
- Score < 0.3: Falls back to generic response

To adjust this threshold, modify the value in `app.py:46`:
```python
if best_score < 0.3:  # Change this value as needed
```

### Custom Responses

The fallback function (`get_fallback` in `app.py`) handles common keywords:
- `track`, `return`, `refund`, `cancel`, `shipping`, `payment`

You can extend this by modifying the `responses` dictionary in the `get_fallback` function.
